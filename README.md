## 🛠️ Technology Stack
* **Cloud Provider:** Microsoft Azure
* **Infrastructure as Code:** Bicep, ARM Templates

* **Security:** Azure Key Vault, Private Endpoints, RBAC

## 🧠 Why I Built This Repository?
**Why? :** To demonstrate a hands-on understanding of cloud architecture. Instead of just using the GUI, I focused on **CLI-first automation** to ensure that every infrastructure piece is repeatable, scalable, and secure. This repository serves as a technical documentation of my transition from a student to a Cloud Professional.
